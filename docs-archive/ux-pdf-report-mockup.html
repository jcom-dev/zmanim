‚òÄÔ∏è Glossary: Primitives

Primitives are foundational astronomical events used in zmanim calculations. These are the building blocks from which all zmanim are derived.

‚òÄÔ∏è sunrise

The moment when the upper edge of the sun's disc crosses the horizon, accounting for atmospheric refraction (~0.833¬∞).

Method: NOAA Solar Position Algorithm
Source: Jean Meeus "Astronomical Algorithms"
üåô sunset

The moment when the upper edge of the sun's disc disappears below the horizon, accounting for atmospheric refraction.

Method: NOAA Solar Position Algorithm
Source: Jean Meeus "Astronomical Algorithms"
üïõ solar_noon

The moment when the sun reaches its highest point (zenith) in the sky for this location. Also known as chatzos.

Method: Solar transit calculation
Source: US Naval Observatory
üåÑ visible_sunrise

Sunrise adjusted for elevation - when an observer at the given elevation first sees the sun appear.

Method: Elevation-adjusted solar calculation
Factor: ~1.15 minutes per 100m elevation

‚öôÔ∏è Glossary: Functions

Functions perform operations on primitives and other values to calculate specific zmanim. They transform base astronomical data into halachically meaningful times.

f(x) solar()

Returns the time when the sun is at a specific angle relative to the horizon. Positive = above, negative = below.

solar(angle, reference_point)
Parameters:
‚Ä¢ angle: degrees (-90 to 90)
‚Ä¢ reference_point: before_sunrise, after_sunset, etc.
f(x) proportional_hours()

Calculates seasonal hours (sha'os zmanios) based on the day length. Each hour is 1/12 of daylight.

proportional_hours(count, framework)
Parameters:
‚Ä¢ count: number of hours (e.g., 3 for Shema)
‚Ä¢ framework: gra, mga, mga_72, etc.
f(x) + / - (arithmetic)

Add or subtract fixed time durations from any zman or primitive.

primitive + 30min, sunrise - 72min
Units: min (minutes), hr (hours), sec (seconds)
f(x) first_valid()

Returns the first non-null result from a list of calculations. Useful for fallback formulas.

first_valid(formula1, formula2, ...)
Example: first_valid(solar(-16.1), sunrise - 72min)
